Setup Guide

Up & Running
in 3 Steps

No technical knowledge needed. Just download, enter your details and watch.

01
📥
Download the App

On your Firestick, open the Downloader app from the Amazon App Store. Enter the code below to download StreamVault.

streamvault.tv/get
02
🔑
Enter Your Details

Open StreamVault and enter the Provider Code, username and password that your IPTV provider gave you. That's it.

03
▶
Start Watching

Your full channel list, movies and series load instantly. Live TV, EPG guide, VOD — everything in one place.

📺 Installing on Firestick via Downloader
  1. From your Firestick home screen, go to Find → Search and search for "Downloader"
  2. Install the orange Downloader app and open it
  3. Go to Settings → Allow JavaScript (toggle on)
  4. Go back to Home and enter: streamvault.tv/get
  5. Download and install StreamVault when prompted
  6. If asked about unknown sources — go to Settings and allow it
